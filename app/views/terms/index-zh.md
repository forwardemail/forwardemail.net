### 积分 {#credits}

| 月度正常运行时间百分比       | 服务天数*       |
| ------------------------- | -------------- |
| < 99.99% – >= 99.90%      | 3              |
| < 99.90% – >= 95.00%      | 7              |
| < 95.50%                  | 15             |

\* 您也可以选择不将服务天数添加到账户，而是让我们发放按比例计算的退款。

最低积分金额必须至少为1.00美元才有资格。最高积分上限为30天或客户根据最近一个日历月因停机支付的总金额（以较高者为准）。受停机影响的付费客户必须在停机日期起30天内通过提交[帮助请求](/help)申请积分。积分仅适用于Forward Email主要控制范围内的停机因素。
